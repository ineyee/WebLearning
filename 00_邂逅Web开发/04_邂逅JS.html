四、邂逅JS

1、什么是JS

JS的全称是JavaScript，翻译过来是Java脚本——当初是为了蹭Java的热度，是为网页添加交互的代码。

2、JS是编程语言吗

JS是一门编程语言。

JS是一门高级、解释型编程语言。

JS是一门基于原型、头等函数的编程语言，是一门多范式的编程语言，是一门支持面向对象编程、指令式编程、函数式编程的编程语言。

3、JS文件的扩展名

JS文件的扩展名是.js。

5、JS的三大组成

  • 语言本身
  • DOM操作（Document Object Model）：通过DOM对象来操作文档（即网页）的API
  • BOM操作（Browser Object Model）：通过BOM对象来操作浏览器——文档（即网页）之外所有其它内容——的API
